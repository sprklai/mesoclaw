## Common Commands

### Development

```bash
# Check current directory
pwd

# List files
ls -la

# Find files
find . -name "*.ts" -type f
```

### Git Operations

```bash
# Check status
git status

# View recent commits
git log --oneline -10

# View diffs
git diff
```
