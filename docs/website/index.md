# MesoCLAW Website Content Plan

This plan defines high-impact website copy for MesoCLAW using product direction and capability signals from:
- [openclaw.ai](https://openclaw.ai/)
- [docs.openclaw.ai](https://docs.openclaw.ai/)
- [openclaw.ai/showcase](https://openclaw.ai/showcase)
- [sipeed/picoclaw](https://github.com/sipeed/picoclaw)

## 1. Messaging Objective
Position MesoCLAW as a practical, trustworthy AI agent platform that helps teams deliver real outcomes, not just chat responses.

## 2. Core Audience
- Engineering teams building internal/external AI workflows
- Ops and product teams automating repetitive, high-context tasks
- Organizations needing control (self-hosting, BYOK, approvals, auditability)

## 3. Messaging Pillars
- **From chat to execution:** AI that can use tools and complete tasks.
- **Control and trust:** self-hosting options, safe execution patterns, configurable approvals.
- **Team-ready workflows:** channel integrations, memory, and reusable skills.
- **Scale path:** lightweight starts with room for production growth.

## 5. Writing Guidance for Iteration
- Keep copy outcome-oriented and concrete.
- Prefer proof-backed capability language over generic AI claims.
- Balance technical credibility with business value statements.
- Reuse showcase patterns (incident triage, content ops, meeting summaries) as conversion anchors.
